类欧几里得算法

https://www.luogu.com.cn/blog/Shuchong/qian-tan-lei-ou-ji-li-dei-suan-fa

主要用来解决直线下整点问题。

比如

∑i=0n⌊ai+bc⌋ \sum_{i=0}^n \left\lfloor { \dfrac{ai+b}{c} } \right\rfloor i=0∑n​⌊cai+b​⌋

前置知识

几个简易的转化：

  1. “求和转化”

p=∑i=1p1 p = \sum_{i=1}^p 1 p=i=1∑p​1

  1. “限制转化”

∑i=1n∑j=1if(i,j)=∑j=1n∑i=1nf(i,j)[j≤i]\sum_{i=1}^n \sum_{j=1}^i f(i, j) = \sum_{j=1}^n\sum_{i=1}^n f(i,j) [j \le i] i=1∑n​j=1∑i​f(i,j)=j=1∑n​i=1∑n​f(i,j)[j≤i]

例题

例一

https://www.luogu.com.cn/problem/P5170 该题中的第一部分。

∑i=0n⌊ai+bc⌋=∑i=0n∑j=0⌊ai+bc⌋−11=∑j=0⌊an+bc⌋−1∑i=0n[j<⌊ai+bc⌋] \begin{aligned} & \sum_{i=0}^n \left\lfloor {\dfrac{ai+b}{c} } \right\rfloor \\=& \sum_{i=0}^n \sum_{j=0}^{\lfloor {\frac{ai+b}{c} } \rfloor - 1} 1 \\=& \sum_{j=0}^{\lfloor \frac{an+b}{c} \rfloor - 1} \sum_{i=0}^n\left[j < \left\lfloor {\dfrac{ai+b}{c}} \right\rfloor \right] \end{aligned} ==​i=0∑n​⌊cai+b​⌋i=0∑n​j=0∑⌊cai+b​⌋−1​1j=0∑⌊can+b​⌋−1​i=0∑n​[j<⌊cai+b​⌋]​

我们化简一下约束的不等式：

j<⌊ai+bc⌋⇒j+1≤ai+bc⇒cj+c≤ai+b⇒cj+c−1<ai+b⇒i>cj+c−b−1a⇒i>⌊cj+c−b−1a⌋\begin{aligned} & j < \left\lfloor {\dfrac{ai+b}{c}} \right\rfloor \\\Rightarrow& j+1\le \dfrac{ai+b}{c} \\\Rightarrow& cj+c\le ai+b \\\Rightarrow& cj+c-1< ai+b \\\Rightarrow& i > \dfrac{cj+c-b-1}{a} \\\Rightarrow& i > \left\lfloor { \dfrac{cj+c-b-1}{a} } \right\rfloor \end{aligned} ⇒⇒⇒⇒⇒​j<⌊cai+b​⌋j+1≤cai+b​cj+c≤ai+bcj+c−1<ai+bi>acj+c−b−1​i>⌊acj+c−b−1​⌋​

把原式中的限制改成对 iii 的限制，有

=∑j=0⌊an+bc⌋−1∑i=0n[i>⌊cj+c−b−1a⌋]=∑j=0⌊an+bc⌋−1∑i=⌊cj+c−b−1a⌋+1n1=∑j=0⌊an+bc⌋−1(n−⌊cj+c−b−1a⌋)=⌊an+bc⌋⋅n−∑j=0⌊an+bc⌋−1⌊cj+c−b−1a⌋\begin{aligned} =& \sum_{j=0}^{\lfloor \frac{an+b}{c}\rfloor -1}\sum_{i=0}^n \left[ i > \left\lfloor \dfrac {cj+c-b-1}{a} \right\rfloor\right] \\=& \sum_{j=0}^{\lfloor \frac{an+b}{c}\rfloor-1}\sum_{i=\lfloor \frac{cj+c-b-1}{a}\rfloor+1}^n 1 \\=& \sum_{j=0}^{\lfloor \frac{an+b}{c}\rfloor-1} \left( {n - \left\lfloor \dfrac {cj+c-b-1}{a} \right\rfloor } \right) \\=& \left\lfloor {\dfrac{an+b}{c}} \right\rfloor \cdot n - \sum_{j=0}^{\lfloor \frac{an+b}{c}\rfloor-1} {\left\lfloor \dfrac {cj+c-b-1}{a} \right\rfloor} \end{aligned} ====​j=0∑⌊can+b​⌋−1​i=0∑n​[i>⌊acj+c−b−1​⌋]j=0∑⌊can+b​⌋−1​i=⌊acj+c−b−1​⌋+1∑n​1j=0∑⌊can+b​⌋−1​(n−⌊acj+c−b−1​⌋)⌊can+b​⌋⋅n−j=0∑⌊can+b​⌋−1​⌊acj+c−b−1​⌋​

我们发现，令 n′=⌊an+bc⌋−1n'=\lfloor \frac{an+b}{c}\rfloor-1n′=⌊can+b​⌋−1，a′=ca'=ca′=c，b′=c−b−1b'=c-b-1b′=c−b−1，c′=ac'=ac′=a，则后半部分为

∑j=0n′⌊a′j+b′c′⌋\sum_{j=0}^{n'} \left\lfloor {\dfrac{a'j+b'}{c'}} \right\rfloor j=0∑n′​⌊c′a′j+b′​⌋

于是可以递归求解。
