离散对数I（BSGS）

P3846 [TJOI2007] 可爱的质数/【模板】BSGS

BSGS(Baby-step Giant-step，大步小步，北上广深，拔山盖世)算法，是用来求解高次同余方程 ax≡b(modn),a⊥na^x\equiv b\pmod n,a\perp nax≡b(modn),a⊥n，或者换个写法 x≡log⁡ab(modn)x\equiv\log_ab\pmod nx≡loga​b(modn)，即离散对数。

前置知识：欧拉定理：aφ(n)≡1(modn),a⊥na^{\varphi(n)}\equiv 1\pmod n,a\perp naφ(n)≡1(modn),a⊥n.

我们根据欧拉定理，ax mod na^x\bmod naxmodn的值每经过 φ(n)\varphi(n)φ(n) 就会有个循环，所以理论上我们只要看看 x∈[1,φ(n)]x\in[1,\varphi(n)]x∈[1,φ(n)] 内有没有 xxx 满足 ax≡ba^x\equiv bax≡b。

其实也并不一定非得是 φ(n)\varphi(n)φ(n) 吧，检验的范围再大一点也不影响结果，为了省去这个求 φ(n)\varphi(n)φ(n) 的过程 分解因数怪麻烦的 ，我们直接检验 x∈[1,n−1]x\in [1,n-1]x∈[1,n−1] 中有没有解。这个算法的时间复杂度是 O(n)\mathcal O(n)O(n) 的。 而使用BSGS算法可以优化到 O(n)\mathcal O(\sqrt n)O(n​)。

算法采用一种分块解决的思想，这也是 “大步” “小步” 的由来。下面是算法的推导过程。

ax≡b(modn)a^x\equiv b\pmod n ax≡b(modn)

设置一个参数 kkk ，把 xxx 写成一种稍有改变的“带余除法”的形式。

x=pk−q(1≤p≤⌊nk⌋+1,1≤q≤k)x=pk-q(1\le p\le\lfloor \dfrac{n}{k}\rfloor+1,1\le q \le k) x=pk−q(1≤p≤⌊kn​⌋+1,1≤q≤k)

这样 xxx 的值域可以包含 [0,n−1][0,n-1][0,n−1] 的整个区间。

于是

apk−q≡b(modn)a^{pk-q}\equiv b\pmod n apk−q≡b(modn)

apk≡baq(modn)a^{pk}\equiv ba^q\pmod n apk≡baq(modn)

把这个式子分成左右两部分，先枚举 q∈[1,k]q\in[1,k]q∈[1,k]，计算处所有的baq mod nba^q\bmod nbaqmodn，存入一个HASH表中。接下来，只要再枚举 p∈[1,⌊nk⌋+1]p\in[1,\lfloor \dfrac{n}{k}\rfloor+1]p∈[1,⌊kn​⌋+1] ，计算 apka^{pk}apk，在HASH表中查找是否存在这样的 qqq 满足上面那个等式即可。如果有多个 qqq，应选择最大的那个，这样可以使得 x=pk−qx=pk-qx=pk−q 最小。

时间复杂度为 O(k+⌊nk⌋)\mathcal O(k + \lfloor \dfrac{n}{k}\rfloor)O(k+⌊kn​⌋)，显然当 k=nk=\sqrt nk=n​ 时有最小复杂度为 O(n)\mathcal O(\sqrt n)O(n​)。写哈希表或者写 map 都可以，差距不大。

	int hs(int x) {
		return x % BS;
